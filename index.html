The Challenge

What we were asked to solve

The NASA Space Apps Challenge is a global hackathon held simultaneously in cities worldwide. In 2023, our team built a solution for the "STAR: Revolutionizing Technical Standards" challenge — developing tools to help organizations navigate, interpret, and apply complex technical standards documentation using AI.

Our Solution

AI-driven standards intelligence

We built an AI-powered system that ingests technical standards documents and makes them searchable using natural language. Engineers could ask questions in plain language and receive relevant sections, context, and related requirements — a RAG-style retrieval system before the term was widely used.

This project directly inspired my current AI.SPIRE focus on Retrieval-Augmented Generation (RAG) and knowledge graph systems — the same core ideas, now applied with far greater technical depth.
Timeline

From brief to 2nd place in 48 hours

Day 0 — Challenge Selection
Team formed, challenge brief analyzed. Identified the core problem: technical standards are inaccessible and hard to cross-reference.
Day 1 — Build Sprint
Built the NLP pipeline for document ingestion, developed the query interface, and implemented cross-reference detection logic.

Finalized the demo, prepared the presentation, and rehearsed the pitch for judges.
